This directory contains all the necessary components to compute alignment distances for multiple examples within a specific domain. To begin, provide a file of examples formatted like cd01068_examples.csv. You can launch the process using the job file. Each test (i.e., each row in the example file) will be processed and stored in a corresponding subdirectory under cd01068_tests. Important: the example file name and the directory name must match.

Once all tests are completed, the program generates a distances.csv file located at full_test_example/cd01068_tests/cd01068_distances. This file contains all five computed distances for each test and can be used for further analysis. Particularly, to compare results with another distances.csv generated for the same domain using a different method. The gap penalties can be adjusted using parms.csv. Note, the subdirectory domain_tests/domain_distances is required to be in the same directory as the testing_code.py and domain_examples.csv. The output you should get for cd01068_examples.csv can be found in the output directory.
